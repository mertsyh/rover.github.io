# İKÜ Roverium Website

This project is a **promotional and content-focused website** developed for the **İKÜ Roverium student organization**. The website aims to present the community, its activities, and projects in a clear and accessible way.

## Project Overview
The website was developed to support the digital presence of the student community by providing:
- Informational content about the organization
- A clean and structured layout
- Accessibility across different devices

The project focuses on **frontend web development fundamentals** and responsive design principles.

---

## Technologies Used
- **HTML5**
- **CSS**
- **JavaScript**

---

## Features
- Responsive and mobile-friendly design  
- Clean and simple user interface  
- Static content structure suitable for community promotion  
- Cross-browser compatibility
